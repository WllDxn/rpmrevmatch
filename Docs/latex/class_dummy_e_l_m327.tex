\doxysubsection{Detailed Description}
A mock ELM327 implementation for testing purposes. 

Simulates ELM327 data by generating random or predefined RPM and speed values, useful for testing the \doxylink{class_gear_box}{Gear\+Box} class without a real ELM327 device.
