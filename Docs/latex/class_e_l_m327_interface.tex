\doxysubsection{Detailed Description}
Implementation of \doxylink{class_e_l_m327_base}{ELM327\+Base} for real ELM327 OBD-\/\+II device communication. 

Communicates with an ELM327 device over a serial port to retrieve vehicle data such as RPM and speed using OBD-\/\+II commands.
