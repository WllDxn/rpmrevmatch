Detailed Description

Implementation of an OBD-II ELM327 interface for calculating rev-matching RPMs based on vehicle speed and gear ratios.

Author
William Dixon
Date
2025-07-22

This program interfaces with an ELM327 OBD-II device (or a dummy simulator) to retrieve vehicle RPM and speed data, processes it through a gearbox model to calculate rev-matching RPMs for downshifts, and logs the results to a CSV file. It supports both real hardware and test modes with configurable parameters.
